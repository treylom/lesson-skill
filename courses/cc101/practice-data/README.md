# practice-data 안내

이 폴더는 CC101 코스의 실습 데이터 모음이다.
비개발자 학습자도 부담 없이 다룰 수 있도록, 숫자 데이터 1개와 문서 데이터 1개로 구성되어 있다.

## 구성 파일
- `sales_data.csv` — 2025년 12개월 매출 데이터 예시
- `meeting_notes.md` — 영업, 생산, 운영, 분기 리뷰 회의록 3건
- `README.md` — 현재 문서

## 학습자가 이 폴더로 해볼 수 있는 것
1. 폴더 탐색 질문하기
2. CSV 열 구조와 패턴 파악하기
3. 회의록에서 반복 이슈 찾기
4. 숫자 데이터와 현장 맥락 연결하기
5. 보고서 초안 만들기

## 추천 질문 예시
- "이 폴더에 어떤 파일이 있고, 무엇부터 보면 좋나요?"
- "sales_data.csv에서 눈에 띄는 패턴 3가지를 찾아주세요."
- "meeting_notes.md에서 반복되는 운영 문제를 정리해주세요."
- "두 파일을 연결해서 보고서 가설을 세워주세요."

## 실습 팁
- 처음부터 완벽한 분석을 하려 하지 않는다.
- 먼저 파일 역할을 파악하고, 그다음 질문 범위를 좁힌다.
- 회의록은 숫자의 이유를 설명하는 맥락 자료로 보면 좋다.
- 보고서는 "관찰 → 해석 → 제안" 순서로 정리하면 쉽다.
